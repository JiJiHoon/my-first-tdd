- [ ] 실행 환경
	- [ ] 컴퓨터가 3개의 숫자를 랜덤하게 선택하고 플레이어가 console을 이용하여 맞춘다.
	- [x] 시작 메시지 출력
	- [ ] 모드 선택
		- [ ] 게임 시작 (play)
		- [ ] 재시작 (replay)
		- [ ] 종료 (quit)

- [ ] 게임 룰
	- [ ] 해당 숫자가 존재하면 1 B (ball)
	- [ ] 해당 숫자가 존재하고, 위치도 일치하면 1 S (strike)
	- [ ] 3개의 숫자가 모두 존재하지 않으면 1 Out
	- [ ] 3 S 이면 승리
	- [ ] 3 Out이면 패배
